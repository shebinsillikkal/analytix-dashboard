# AnalytiX — Real-Time Analytics Dashboard

> A SaaS analytics dashboard where data updates every second — no more emailing spreadsheets around.

## The Problem
A client wanted their whole team looking at the same live numbers without manually refreshing reports or emailing spreadsheets. Built this so data appears in real time on everyone's screen simultaneously.

## Features
- 📊 40+ chart types — line, bar, scatter, heatmap, funnel, geo maps
- 🔄 Live WebSocket data feeds — sub-second refresh
- 🧩 Drag-and-drop widget layout — each user customises their view
- 👥 Multi-tenant — each company gets their own workspace
- 🔐 RBAC — admins, editors, viewers with different permissions
- 📤 Export to PDF/PNG, scheduled email reports
- 📱 Fully responsive — works on phone & tablet

## Stack
```
React 18 | TypeScript | D3.js | WebSocket | TailwindCSS | Node.js | PostgreSQL | Redis
```
